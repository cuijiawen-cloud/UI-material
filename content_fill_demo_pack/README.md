## 2. 匹配规则

- 系统根据内容区文案匹配 display_name 或 aliases。
- 内容区出现“角色A”或“A”时，匹配 assets/A.png。
- 内容区出现“角色B”或“B”时，匹配 assets/B.png。
- 内容区出现“角色C”或“C”时，匹配 assets/C.png。
- 内容区出现“材料D”或“D”时，匹配 assets/D.png。
- 内容区出现“材料E”或“E”时，匹配 assets/E.png。
- 内容区出现“材料F”或“F”时，匹配 assets/F.png。
- 角色内容只能匹配 character 类型素材。
- 材料内容只能匹配 material 类型素材。
- 没有匹配到素材时，不要自行猜测。
- 不要修改内容区原文案。

## 3. 注意事项

- 本包仅用于内容填充 demo。
- 本次只做重命名、打标和打包。
- 不要修图、抠图、裁切、生成新图。
- 不要根据图片外观猜测真实角色名或材料名。

## 4. 缺失说明

- 本次指定素材均已提供，无缺失素材。
